# NEOVOlTaa Website

Современный корпоративный сайт для ООО NEOVOLT.

## Технологии

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- TypeScript

## Установка

1. Установите зависимости:
```bash
npm install
# или
yarn install
```

2. Запустите сервер разработки:
```bash
npm run dev
# или
yarn dev
```

3. Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере.

## Сборка для продакшена

```bash
npm run build
# или
yarn build
```

После сборки запустите продакшен-сервер:

```bash
npm run start
# или
yarn start
```
